🚀 HOW TO RUN ScanOps 🔍

This tool scans all ports of a given domain/IP and shows useful IP info.

--------------------------------------------
🔧 Requirements:
- Python 3.7 or higher
- Internet connection (for IP info API)
--------------------------------------------

📦 Step 1: Install required libraries

Open your terminal or command prompt and run:

    pip install -r requirements.txt

--------------------------------------------

⚙️ Step 2: Run the tool

In the terminal, navigate to the folder where ScanOps.py is located.

Then run:

    python ScanOps.py

--------------------------------------------

🌐 Step 3: Enter a Target

You will be prompted to enter a domain or IP address. For example:

    🌐 Enter domain or IP: example.com

The tool will:
- ✅ Resolve the domain to IP
- 🌐 Fetch IP info (country, ISP, city, etc.)
- 🔍 Scan all 65,535 ports
- 📊 Show a progress bar
- ✅ List open ports (if any)

--------------------------------------------

❌ To stop the scan anytime, press:
    CTRL + C

Enjoy scanning responsibly! 👨‍💻
